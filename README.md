# Bluetooth Controlled Sumo Robot

## Overview
This project is a Bluetooth-controlled sumo robot developed using Arduino. The robot is designed to maneuver within a sumo arena and push opponent robots out using remote control and motor-driven movement.

## Components Used
- Arduino Uno
- L298N Motor Driver
- DC Motors
- Robot Chassis
- Battery Pack

## Features
- Bluetooth-based wireless control
- Forward, backward, left and right movement
- Motor speed control
- Sumo robot competition design

## Applications
- Robotics competitions
- Arduino learning projects
- Wireless robotic control systems
